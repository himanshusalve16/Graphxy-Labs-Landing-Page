Lumen

A calm, glass-and-light visual language. Inter carries the interface; Fraunces carries the tutor's voice. Color identifies the subject lens, not the state of the UI — accents are reserved for math, chemistry, biology, and physics.

Display · Fraunces 500 · 26px — section headings within explanations

Why the graph curves the way it does

Body · Inter 400 · 17px — explanation summary

Multiplying x² by a larger value of a stretches the graph vertically. Every point gets pulled further from the x-axis, so the curve appears to rise more steeply — which reads visually as "narrower," even though the shape is still a parabola.

Iconography & Misc

Icons: lucide-react — thin-stroke line icons (2px), matching the restrained, neutral chrome. Reserve color for subject tags and accent states only; icons stay var(--text-secondary) by default. Corner radii: 10px (small controls), 16px (inputs/buttons), 24px (cards/canvas). Shadows are always soft and layered — never a single hard drop shadow.

Visual Language & Philosophy

Graphxy Labs operates at the intersection of rigorous science, mathematical clarity, and high-performance software engineering. Our design aesthetic reflects this philosophy: it is minimalist, clean, highly structured, and utilizes subtle grid lines, micro-animations, and vibrant, contextual color indicators.

The primary design principles are precision, symmetry, and interaction. Visual modules are framed in light cards, set against a warm, off-white background, and brought to life using interactive animated SVGs and physics-based motion.

Motion & Interaction Principles

Animations in Graphxy Labs must always feel organic, deliberate, and high-fidelity. We do not use standard linear curves; all movement adheres to physics-based deceleration curves.

Animation Type Timing Parameter Duration Description / Application
Interactive Spring Scale spring(300, 15) Dynamic Used on node hover (desktop) or active clicks to give springy elastic feel.
Standard Deceleration cubic-bezier(0.16, 1, 0.3, 1) 200ms - 300ms Used on accordions expand, hover borders transition, tab switching, and panel slides.
Living Pulses animate-pulse 3s (linear, infinite) Used on central parent node outline waves to denote active live calculations.
Data Particle Trails animate={ cx: [x1, x2] } 3.5s - 4.5s Infinite linear loops on lines representing data traveling between enclaves.

Accessibility Rules

  • WCAG Touch Targets: Ensure that all clickable nodes and menu items have a minimum hit area of 44x44px (using transparent `circle` or `rect` overlays).
  • Contrast Ratios: Text elements must maintain a minimum contrast ratio of 4.5:1 against the white or off-white background. No faint gray text is allowed on white surfaces.
  • iOS Zoom Prevention: Inputs, textareas, and select components must utilize a font-size of at least 16px to prevent automated iOS viewport zooms on focus.
  • Touch Delay Elimination: Set touch-action: manipulation; on all button, link, and trigger elements to remove the iOS 300ms tap delay.

# Graphxy Labs — Document Repository

Welcome to the central documentation repository for Graphxy Labs. This repository acts as the single source of truth for our product designs, system architectures, research studies, and intellectual property.

## Repository Layout
- [/docs](file:///d:/Graphxy/docs) — High-level corporate and product portfolios
- [/product](file:///d:/Graphxy/product) — Comprehensive Product Requirements Documents (PRDs)
- [/design](file:///d:/Graphxy/design) — CSS design systems and visually rich HTML templates
- [/wireframes](file:///d:/Graphxy/wireframes) — Site maps, information architectures, and user journeys
- [/screens](file:///d:/Graphxy/screens) — Comprehensive UI inventories and layout trees
- [/architecture](file:///d:/Graphxy/architecture) — Frontend, backend, and calculation engines
- [/research](file:///d:/Graphxy/research) — Competitive landscapes and prior art analyses
- [/ip](file:///d:/Graphxy/ip) — Novelty summaries and inventor logs
- [/patent](file:///d:/Graphxy/patent) — Draft claims and patent readiness disclosures
- [/assets](file:///d:/Graphxy/assets) — Design references, screenshots, and patent drawings

For onboarding developers, designers, or legal advisors, start with the corporate portfolios in the `/docs` directory.
